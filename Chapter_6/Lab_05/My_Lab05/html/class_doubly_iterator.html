Detailed Description

template<typename T>
class DoublyIterator< T >

연결리스트에서 쓰이는 Iterator의 클래스. 참조만 하는 것이 아니라 수정이 가능하게끔 하여 기존의 iterator 처럼 사용 const가 아니다.
